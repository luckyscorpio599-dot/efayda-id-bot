# EFAYDA Telegram ID Bot

This bot generates Ethiopian ID cards from EFAYDA PDFs.

## Features
- Works for any user PDF
- Crops photo, QR, barcode
- Extracts all text fields dynamically
- Generates a single combined ID image
- Deployable on Render & GitHub
